   - Add `visibility` to the select fields: `visibility: collectionItems.visibility`
   - Add `audioFormat: collectionItems.audioFormat`, `bitrate: collectionItems.bitrate`, `sampleRate: collectionItems.sampleRate`
   - In the row mapping at the bottom, set `openForTrade` from visibility for backward compat:

Note: The current `getCollectionPage` has a workaround that hardcodes `openForTrade: 0` because the migration may not have been applied. Remove this workaround — the Plan 01 migration adds visibility with a default, so the column will always exist after this phase.
