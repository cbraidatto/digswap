# Phase 7 — UI Design Contract

> Visual and interaction contract for Community + Reviews. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn |

## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding, genre chip internal gap, star character spacing |
| sm | 8px | Compact element spacing, chip row gap, group card gap, post separator gap |
| md | 16px | Default element spacing, group card padding, post card padding, form field spacing |
| lg | 24px | Section padding, group detail header vertical padding, page section gap |
| xl | 32px | Page-level horizontal padding (mobile), group discovery section gap |
| 2xl | 48px | Not used in Phase 7 |
| 3xl | 64px | Not used in Phase 7 |

Exceptions:
- Touch targets for join/leave buttons: 44px minimum height (accessibility)
- Separator line (post card `─────`): 0 height, purely visual

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#10141a` (--background/--surface) | Page background, main content area |
| Secondary (30%) | `#181c22` (--card/--surface-container-low) | Group cards, post cards, group detail background, form fields |
| Accent (10%) | `#6fdd78` (--primary) | See reserved list below |
| Destructive | `#ffb4ab` (--destructive) | Leave group confirmation, error states only |

*Source: globals.css, Phase 6 UI-SPEC (inherited unchanged)*

**Accent (`#6fdd78` / --primary) reserved for:**
- Active filter chip (genre chips) background at 10% opacity (`bg-primary/10 text-primary border-primary`)
- Terminal prompt character `>` in composer input
- `[+ CREATE_GROUP]` button border and text
- Active tab indicator on group detail page (if tabbed)
- Join button primary state border and text
- Record title hover color in post linked record row
- Active/joined group indicator `[JOINED]` text

**Secondary accent (`#aac7ff` / --secondary) reserved for:**
- "Rare" rarity badge on linked record in posts (reuses existing badge system)
- Group invite accepted notification type icon tint

**Tertiary (`#ffb689` / --tertiary) reserved for:**
- "Ultra Rare" rarity badge on linked records (reuses existing badge system)
- Star characters in review cards (`color: var(--tertiary)`)

**Structural colors:**
- Post separator line `─────`: `text-on-surface-variant/20` (barely visible, purely structural)
- `[PRIVATE]` badge: `bg-surface-container-high text-on-surface-variant border border-outline-variant/20`
- `[INVITE_ONLY]` label: `text-on-surface-variant` (muted, no interaction)
- `[JOINED]` indicator: `text-primary` (accent, confirms membership)
- Group card hover: `bg-surface-container` (one step above card background)
- Review card elevated from post card: `bg-surface-container` vs post `bg-surface-container-low`

### Group Composer

**Composer placement:** Below the group header, above the post feed. Always visible to members.

```
+-------------------------------------------------------+
| [ Write a post...                               ]     |
+-------------------------------------------------------+
| [+ link record]              [Write review] [Post]    |
+-------------------------------------------------------+
```

**Normal post mode:**
- Textarea: `bg-surface-container-low border border-outline-variant/20 rounded p-3 w-full resize-none`
- Placeholder: `Write a post...` -- Inter, 14px, `text-on-surface-variant`
- Minimum height: 80px (2-3 lines)
- `[+ link record]` button: JetBrains Mono, 10px, `text-on-surface-variant border border-outline-variant/20 px-2 py-1 rounded hover:text-primary hover:border-primary`
- `[Write review]` button: JetBrains Mono, 10px, same style as `[+ link record]`
- `[Post]` button: JetBrains Mono, 10px, `bg-primary text-primary-foreground px-4 py-1 rounded`, disabled when textarea empty

**Review mode (activated by `[Write review]`):**
- Star rating selector appears above textarea: 5 clickable star characters `★★★★★`
  - Unselected: `text-on-surface-variant/40`
  - Hover highlight: `text-tertiary/80`
  - Selected (filled): `text-tertiary`
  - Clicking a star sets rating (1-5), clicking again on same star clears it
- Record link becomes required (indicator: `* RECORD_REQUIRED` in JetBrains Mono, 10px, destructive if not linked on submit)
- Submit button changes to `[Post Review]`
- Composer mode label: `· review mode ·` in JetBrains Mono, 10px, `text-on-surface-variant`

**Record search (triggered by `[+ link record]`):**
- Opens a shadcn Dialog containing the same search input pattern as Phase 6 (RecordSearchInline)
- Terminal prompt `>` in search input, green left border
- Results list: title + artist + year, clicking selects the record and closes dialog
- Selected record appears as a compact token below the textarea:
  - `└ {Artist} - {Title} [{Label}, {Year}]` -- JetBrains Mono, 10px, `text-on-surface-variant`
  - `[x]` remove button: 12px `text-on-surface-variant hover:text-destructive`

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| Join/Leave button | `aria-label` reflects current state: `aria-label="Join {group name}"` / `aria-label="Leave {group name}"` |
| Star rating selector (composer) | `role="radiogroup"` with 5 `role="radio"` buttons, `aria-label="Rating: {N} out of 5 stars"` |
| Star display (read-only) | `aria-label="{N} out of 5 stars"` on the span containing star characters |
| Review expand trigger | `aria-expanded` toggles true/false, `aria-controls` points to panel id |
| Post feed list | `role="list"` on feed container, each post `role="listitem"` |
| Group discovery list | `role="list"` on member group section, each card `role="listitem"` |
| Genre filter chips | Each chip `<button>` with `aria-pressed` reflecting active state |
| Composer textarea | `aria-label="Write a post"`, `aria-required` in review mode |
| Group creation form | All inputs have associated `<label>` elements (not just visual labels) |
| `[PRIVATE]` badge | `aria-label="Private group"` |
| Focus management | After joining: focus moves to composer textarea. After leaving: focus returns to join button area. |
| Color contrast | `#6fdd78` on `#10141a` = 9.7:1 (WCAG AAA). `#ffb689` (tertiary/stars) on `#10141a` = 8.1:1 (WCAG AAA). `#ffb4ab` on `#10141a` for errors = 9.5:1 (WCAG AAA). |
| Reduced motion | All animations respect `prefers-reduced-motion: reduce` (inherited from globals.css) |
| Keyboard navigation | All interactive elements reachable by Tab. Enter activates buttons. Escape closes composer dialog and leave confirmation. |
