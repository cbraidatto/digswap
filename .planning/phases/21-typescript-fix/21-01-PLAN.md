```typescript
for (const row of rows as Array<{ gem_tier: string; count: string }>) {
```
to:
```typescript
for (const row of rows as unknown as Array<{ gem_tier: string; count: string }>) {
```

```typescript
const result = rows as Array<{ gem_score: string | null }>;
```
to:
```typescript

```

Both fixes cast through `unknown` first, which is the standard TypeScript pattern for bridging non-overlapping types from raw SQL query results.
