    For each of the 13 files, read the file first, identify all exported `"use server"` functions that mutate data or call external APIs, and add rate limiting after the auth check. Pattern:

    // Inside each server action, AFTER the auth check:

    - `collection.ts` -- use `apiRateLimit` (30/60s)
