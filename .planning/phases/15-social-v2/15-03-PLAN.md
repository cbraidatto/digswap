## Objective

Add live sync (new messages via Supabase Realtime) and presence indicators (read-only, derived from desktop heartbeat) to the trade detail page. Web is a passive observer of presence — the desktop is the authority.

## Context

`trade_runtime_sessions` has:
- `user_id` — who holds the session
- `trade_id` — which trade
- `heartbeat_at` — last heartbeat timestamp
- `released_at` — null if active

Two participants in a trade: `requester_id` and `provider_id` from `trade_requests`. Presence = session exists, released_at IS NULL, heartbeat_at within 2 minutes.

RLS on `trade_runtime_sessions` blocks direct client reads (Phase 17 design decision — only RPCs can query). Need a read-safe RPC or query via service role on the server side.

If the existing RPC `acquire_trade_lease` / `heartbeat_trade_lease` don't expose a read path, add a minimal `get_trade_presence` RPC:

```sql
create or replace function public.get_trade_presence(p_trade_id uuid)
returns table (user_id uuid, heartbeat_at timestamptz)
language sql security definer
set search_path = public
as $$
  select user_id, heartbeat_at
  from public.trade_runtime_sessions
  where trade_id = p_trade_id
    and released_at is null
    and heartbeat_at > (now() - interval '2 minutes')
    and exists (
      select 1 from public.trade_requests tr
      where tr.id = p_trade_id
        and (tr.requester_id = auth.uid() or tr.provider_id = auth.uid())
    );
$$;
```

Add this to a new migration `supabase/migrations/20260401_trade_presence_rpc.sql` if needed.

### Task 3 — Live message sync in TradeMessageThread

Update `TradeMessageThread` to subscribe to new `trade_messages` inserts via Supabase Realtime:

```typescript
// In TradeMessageThread client component:
useEffect(() => {
  const channel = supabase
    .channel(`trade-messages:${tradeId}`)
    .on('postgres_changes', {
      event: 'INSERT',
      schema: 'public',
      table: 'trade_messages',
      filter: `trade_id=eq.${tradeId}`,
    }, (payload) => {
      setMessages(prev => [...prev, payload.new as TradeMessage]);
      // auto-scroll to bottom
    })
    .subscribe();

  return () => { void supabase.removeChannel(channel); };
}, [tradeId]);
```
