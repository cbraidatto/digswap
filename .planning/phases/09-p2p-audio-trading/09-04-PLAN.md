**Create src/app/(protected)/trades/[id]/_components/transfer-progress.tsx** — Client component ("use client"):
Per UI-SPEC TransferProgress Specification:

Props:
```typescript
interface TransferProgressProps {
  progress: number;       // 0-100
  bytesTransferred: number;
  totalBytes: number;
  speed: number;          // bytes/sec
  eta: number;            // seconds remaining
}
```

Layout per UI-SPEC:
- Container: `bg-surface-container-low rounded-lg p-4 border border-outline-variant/10 w-full`
- Percentage display above bar: `font-mono text-2xl font-bold font-heading text-primary` — e.g., "67%"
- Progress bar: `h-2 bg-surface-container-high rounded-full overflow-hidden`
- Fill: `h-full bg-primary rounded-full transition-all duration-300` with `width: ${progress}%` inline style
- Progress bar uses `role="progressbar"` with `aria-valuenow={progress}`, `aria-valuemin={0}`, `aria-valuemax={100}` (UI-SPEC accessibility)
- Stats row below bar: `font-mono text-[10px] text-on-surface-variant` with dot separators
- Format: `{transferred} / {total} . {speed}/s . ~{eta} remaining`
  - transferred/total: formatted via `formatFileSize()` from `@/lib/audio/file-metadata` (e.g., "28.3 MB / 42.3 MB")
  - speed: formatted as MB/s or KB/s depending on magnitude
  - eta: formatted as seconds or minutes (e.g., "~12s remaining" or "~2m remaining")
- Respect `prefers-reduced-motion`: remove `transition-all` when motion is reduced

Helper function `formatSpeed(bytesPerSec: number): string`:
- If >= 1MB/s: `${(bytesPerSec / 1024 / 1024).toFixed(1)} MB`
- Else: `${(bytesPerSec / 1024).toFixed(1)} KB`

Helper function `formatEta(seconds: number): string`:
- If >= 60: `~${Math.ceil(seconds / 60)}m`
- Else: `~${Math.ceil(seconds)}s`
