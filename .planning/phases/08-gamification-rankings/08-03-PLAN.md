**Create `src/app/(protected)/(explore)/explorar/_components/rankings-tab.tsx`:**

Client component (`"use client"`). This is the main leaderboard container.

Import:
- `useState`, `useEffect`, `useCallback` from `react`
- `loadGlobalLeaderboard`, `loadGenreLeaderboard` from `@/actions/gamification`
- `LeaderboardRow` from `./leaderboard-row`
- `GenreFilter` from `./genre-filter`
- `type LeaderboardEntry` from `@/lib/gamification/queries`

Props:
```typescript
interface RankingsTabProps {
  currentUserId: string;
}
```

State:
- `activeGenre: string | null` -- null = GLOBAL, string = genre name
- `entries: LeaderboardEntry[]` -- leaderboard data
- `loading: boolean` -- initial load state
- `loadingMore: boolean` -- pagination loading state
- `page: number` -- current page (starts at 1)
- `hasMore: boolean` -- whether more pages exist

Behavior:
1. On mount and when `activeGenre` changes, reset `page` to 1, set `loading` to true, fetch first page of leaderboard data:
   - If `activeGenre === null`: call `loadGlobalLeaderboard(1)`
   - Else: call `loadGenreLeaderboard(activeGenre, 1)`
2. Set `hasMore` to `entries.length === 50` (page size)
3. "Load More" button calls the same action with `page + 1`, appends results to `entries`

Layout (per UI-SPEC):
```
max-w-4xl mx-auto p-8 md:p-12
```

Structure:
1. `GenreFilter` at top with current `activeGenre` and `onGenreChange` handler (disabled during loading)
2. Section header: `font-mono text-[10px] uppercase tracking-wider text-tertiary mt-6 mb-4`
   - If GLOBAL: `LEADERBOARD_GLOBAL`
   - If genre: `LEADERBOARD_{GENRE.toUpperCase().replace(/[^A-Z0-9]/g, '_')}`
3. `<ol role="list" className="space-y-1">` wrapping `LeaderboardRow` components
   - Each entry: `<LeaderboardRow rank={entry.globalRank} username={entry.username} title={entry.title} score={entry.globalScore} isOwnUser={entry.userId === currentUserId} />`
4. Loading state: 3 skeleton rows (`div className="bg-surface-container-low rounded-lg h-12 animate-pulse"`)
5. Empty state: `<div className="text-center py-12"><p className="font-mono text-[10px] text-on-surface-variant uppercase tracking-wider">NO_RANKINGS_YET</p><p className="font-mono text-[10px] text-outline mt-2">Rankings are computed every 15 minutes. Import your collection and start digging to appear on the leaderboard.</p></div>`
6. Load More button at bottom (if `hasMore && !loading`): `<button className="w-full py-3 font-mono text-[10px] uppercase tracking-wider text-on-surface-variant hover:text-on-surface transition-colors" disabled={loadingMore}>{loadingMore ? "LOADING..." : "LOAD_MORE"}</button>`
