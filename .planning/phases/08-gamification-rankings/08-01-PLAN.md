**Create `src/lib/gamification/badge-awards.ts`:**

Export `awardBadge(userId: string, badgeSlug: string): Promise<boolean>`:
1. Use `createAdminClient()` from `@/lib/supabase/admin`
2. Look up badge by slug: `admin.from("badges").select("id, name").eq("slug", badgeSlug).single()`
3. Check existing award: `admin.from("user_badges").select("id").eq("user_id", userId).eq("badge_id", badge.id).maybeSingle()`
4. If already awarded, return false (idempotent)
5. Insert: `admin.from("user_badges").insert({ user_id: userId, badge_id: badge.id, earned_at: new Date().toISOString() })`
6. Create notification: `admin.from("notifications").insert({ user_id: userId, type: "new_badge", title: "Badge earned: ${badge.name}", body: "You've earned the ${badge.name} badge!", link: "/perfil" })`
7. Return true on success, false on any error
8. Wrap all in try/catch -- never throw

Functions:

1. `getGlobalLeaderboard(page = 1, pageSize = 50): Promise<LeaderboardEntry[]>` -- Drizzle select from `userRankings` inner join `profiles` on userId=profiles.id, compute `globalScore` as `sql<number>\`${userRankings.rarityScore} * 0.7 + ${userRankings.contributionScore} * 0.3\``, order by `asc(userRankings.globalRank)`, limit pageSize offset (page-1)*pageSize.

2. `getGenreLeaderboard(genre: string, page = 1, pageSize = 50): Promise<LeaderboardEntry[]>` -- Drizzle raw SQL query (use `db.execute(sql\`...\`)` or build with Drizzle builders). Join `collection_items` + `releases` + `profiles`, WHERE `releases.genre @> ARRAY[${genre}]::text[]`, GROUP BY user_id, username, display_name. Compute `SUM(ln(1 + COALESCE(releases.rarity_score, 0)))` as `genre_rarity_score`, `ROW_NUMBER() OVER (ORDER BY SUM(...) DESC)` as `genre_rank`. Return rows matching `LeaderboardEntry` shape (set globalScore to genre_rarity_score, globalRank to genre_rank, title from user_rankings join or null). LIMIT pageSize OFFSET (page-1)*pageSize.

3. `getUserRanking(userId: string): Promise<UserRanking | null>` -- Select from `userRankings` where userId matches. Return null if no row.

4. `getUserBadges(userId: string): Promise<UserBadge[]>` -- Select from `userBadges` inner join `badges` on badgeId=badges.id, where userId matches, order by `asc(userBadges.earnedAt)`. Map to UserBadge shape.

5. `getLeaderboardCount(): Promise<number>` -- Count of rows in userRankings table (for pagination info).

6. `getGenreLeaderboardCount(genre: string): Promise<number>` -- Count of distinct users who have records in the given genre.
