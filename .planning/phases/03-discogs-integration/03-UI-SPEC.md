## Page Specifications

### Import Progress Page (`/import-progress`)

**Route:** `(protected)/import-progress/page.tsx`
**Purpose:** Full-screen view of active import (DISC-04, D-04, D-06)
**Protected:** Yes (requires auth)
**Shell:** Renders within the AppShell (header + bottom bar visible), no tab highlighted

#### Layout

```
+-------------------------------------------+
|  AppHeader                                |
+-------------------------------------------+
|                                           |
|  padding-top: 48px (2xl)                  |
|                                           |
|        [Disc3 icon 48px, spinning]        |
|        (accent color, full opacity)       |
|                                           |
|        gap: 32px (xl)                     |
|                                           |
|  +--- Card ----------------------------+ |
|  | padding: 24px (lg)                  | |
|  |                                     | |
|  | "Importing your collection"         | |
|  | Fraunces 20px semibold              | |
|  |                                     | |
|  | [==========--------] 27%           | |
|  | Progress bar, accent fill           | |
|  |                                     | |
|  | 342 / 1,247 records                 | |
|  | DM Sans 14px, muted-foreground      | |
|  |                                     | |
|  | Currently importing:                | |
|  | Kind of Blue -- Miles Davis         | |
|  | DM Sans 14px label + 16px value     | |
|  +-------------------------------------+ |
|                                           |
+-------------------------------------------+
|  BottomBar                                |
+-------------------------------------------+
```

#### States

| State | Visual |
|-------|--------|
| **Importing collection** | Heading: "Importing your collection". Disc3 spins (CSS animation `spin 2s linear infinite`). Progress bar fills from left. Count updates live. Current record updates live. |
| **Importing wantlist** | Heading changes to "Importing your wantlist". Progress bar resets for wantlist items. Same layout. |
| **Success** | Heading: "Import complete!". Disc3 stops spinning, replaced with a static check-circle icon (Lucide `CheckCircle2`, success color). Count shows final total: "1,247 records imported". Body: "Redirecting to your collection...". Auto-redirect to `/perfil` after 2 seconds (D-06). |
| **Error** | Heading: "Import paused". Disc3 stops spinning. Error message shown in destructive color. CTA: "Retry Import" (accent button). Body: "Something went wrong during import. Your progress has been saved -- tap retry to continue where you left off." |
| **Resumed (user returns)** | If user navigates back to this page while import is still running: reconnects to Realtime channel and picks up current progress. Same visual as "Importing collection/wantlist". |

#### Accessibility

- Progress bar: `role="progressbar"`, `aria-valuenow={current}`, `aria-valuemin={0}`, `aria-valuemax={total}`, `aria-label="Import progress"`
- Current record text: `aria-live="polite"` so screen readers announce changes
- Spinning icon: `aria-hidden="true"` (decorative)
- Reduced motion: Disc3 spin animation disabled via `prefers-reduced-motion: reduce` (already in globals.css)

#### Skeleton / Loading

When page loads before Realtime connection establishes:
- Disc3 icon shown immediately (static, not spinning)
- Card shown with skeleton shimmer blocks: 1 skeleton heading line (200px wide), 1 skeleton progress bar, 1 skeleton text line (120px wide), 1 skeleton text line (180px wide)
- Skeleton uses existing `skeleton-shimmer` class from globals.css

### Disconnect Dialog (D-15)

Uses `alert-dialog` (shadcn) for destructive confirmation pattern.

```
+--- AlertDialog -------------------------+
|  AlertDialogHeader                      |
|    Title: "Disconnect Discogs?"         |
|    Description:                         |
|      "Your imported collection and      |
|       wantlist will be removed from     |
|       VinylDig. Records added manually  |
|       will not be affected."            |
|                                         |
|  AlertDialogFooter                      |
|    [Keep Discogs] (outline variant)     |
|    [Disconnect] (destructive variant)   |
+-----------------------------------------+
```

- Title: Fraunces 16px medium (uses existing `DialogTitle` size from dialog.tsx)
- Description: DM Sans 14px muted-foreground
- Keep Discogs: outline variant, auto-closes dialog
- Disconnect: destructive variant, triggers `disconnectDiscogs` server action
- During disconnect: "Disconnect" button shows spinner, disabled. "Keep Discogs" also disabled.
- On success: dialog closes, settings card transitions to disconnected state, toast: "Discogs disconnected"

### Sticky Import Banner (D-09, D-10)

Appears at the top of the content area (below AppHeader) during any active import or sync.

#### Layout

```
+-------------------------------------------+
|  AppHeader                                |
+-------------------------------------------+
| [Disc3 16px spinning] Importing... 342/1,247  [>] |
| Banner: h-44px, bg-secondary, border-b    |
+-------------------------------------------+
|  Content Area                             |
```

#### Specification

- Position: sticky, top 56px (below header), left 0, right 0
- Height: 44px
- Background: `oklch(0.18 0.02 55)` (secondary)
- Bottom border: 1px solid `oklch(0.25 0.02 55)` (border)
- Z-index: 35 (below header at 40)
- Layout: flex row, items-center, gap 8px (sm), padding 0 16px (md)
- Left content: Disc3 icon (16px, accent, spinning), "Importing... 342/1,247" in DM Sans 14px muted-foreground
- Right content: Chevron-right icon (16px, muted-foreground) as navigation affordance
- Entire banner is tappable: navigates to `/import-progress` (D-09)
- Cursor: pointer
- Banner disappears instantly when import completes (no animation)
- Banner reappears if user navigates away and back while import is still running

#### Where it renders

- Inside `(protected)/layout.tsx`, conditionally shown when an active import job exists for the current user
- Appears on ALL protected pages during import, not just Perfil (user might navigate to Feed or Explorar)
- Uses Zustand store to track import active state, hydrated from Supabase Realtime subscription

#### Accessibility

- `role="status"`, `aria-live="polite"` for screen readers
- `aria-label="Import in progress, 342 of 1,247 records. Tap to view details."`
- Tappable area: entire 44px height bar (meets minimum touch target)
