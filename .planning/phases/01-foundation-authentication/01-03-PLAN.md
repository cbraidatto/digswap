  <action>
    1. **src/lib/supabase/client.ts** -- Browser Supabase client (per 01-RESEARCH.md):
       ```typescript
       import { createBrowserClient } from "@supabase/ssr";
       export function createClient() {
         return createBrowserClient(
           process.env.NEXT_PUBLIC_SUPABASE_URL!,
           process.env.NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY!
         );
       }
       ```

    2. **src/lib/supabase/server.ts** -- Server Supabase client (per 01-RESEARCH.md Pattern 1):
       - Import `createServerClient` from `@supabase/ssr`
       - Import `cookies` from `next/headers` (async in Next.js 15)
       - Create client with cookie getAll/setAll handlers
       - Uses NEXT_PUBLIC_SUPABASE_URL and NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY
       - Export as async function `createClient()`

    3. **src/lib/supabase/admin.ts** -- Service role client (bypasses RLS):
       - Import `createClient` from `@supabase/supabase-js`
       - Create with NEXT_PUBLIC_SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY
       - Set `auth: { autoRefreshToken: false, persistSession: false }`
       - Export as `createAdminClient()`
       - WARNING: Never import this on the client side. Never use in client components.

    4. **src/lib/supabase/middleware.ts** -- Middleware helper function:
       - Import `createServerClient` from `@supabase/ssr`
       - Create `updateSession(request)` function following exact pattern from 01-RESEARCH.md Pattern 1
       - CRITICAL SECURITY: Use `supabase.auth.getClaims()` (NEVER `getSession()`) to validate JWT
       - Extract user from claims via `claims?.sub`
       - Redirect unauthenticated users from `/onboarding*` and other protected routes to `/signin`
       - Redirect authenticated users from `/signin`, `/signup` to `/onboarding` (if onboarding not completed) or `/` (if completed)
       - Return the supabaseResponse with updated cookies

    5. **src/middleware.ts** -- Next.js middleware entry point:
       - Import `updateSession` from `@/lib/supabase/middleware`
       - Call `updateSession(request)` and return result
       - Configure matcher to exclude static files: `/((?!_next/static|_next/image|favicon.ico|.*\\.(?:svg|png|jpg|jpeg|gif|webp)$).*)`
  </action>
