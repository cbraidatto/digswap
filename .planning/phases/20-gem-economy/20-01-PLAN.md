    **1. Create `apps/web/src/lib/gems/constants.ts`:**

    Export the GemTier type as a union: `'quartz' | 'amethyst' | 'emerald' | 'ruby' | 'sapphire' | 'diamond'`

    Export the GEM_TIERS array (per CONTEXT.md locked decision):
    ```typescript
    export const GEM_TIERS = [
      { key: 'quartz' as const,   name: 'Quartzo',   maxRatio: 0.3,        weight: 1,   color: '#9CA3AF' },
      { key: 'amethyst' as const, name: 'Ametista',   maxRatio: 0.8,        weight: 3,   color: '#8B5CF6' },
      { key: 'emerald' as const,  name: 'Esmeralda',  maxRatio: 1.5,        weight: 8,   color: '#10B981' },
      { key: 'ruby' as const,     name: 'Rubi',       maxRatio: 3.0,        weight: 20,  color: '#EF4444' },
      { key: 'sapphire' as const, name: 'Safira',     maxRatio: 6.0,        weight: 35,  color: '#3B82F6' },
      { key: 'diamond' as const,  name: 'Diamante',   maxRatio: Infinity,   weight: 100, color: '#F0F9FF' },
    ] as const;
    ```

    Export `getGemTier(score: number | null): GemTier | null` — maps ratio to tier using thresholds: <0.3 quartz, <0.8 amethyst, <1.5 emerald, <3.0 ruby, <6.0 sapphire, >=6.0 diamond. Returns null for null/undefined input.

    Export `getGemWeight(tier: GemTier): number` — returns the weight for a tier (1, 3, 8, 20, 35, 100).

    Export `getGemInfo(tier: GemTier)` — returns the full tier object from GEM_TIERS.

    Export `computeGemScore(scores: number[]): number` — takes an array of rarity_score values, maps each to its gem tier weight, and returns the sum.

    **2. Create `apps/web/src/lib/gems/queries.ts`:**

    Export `getGemDistribution(userId: string): Promise<Record<GemTier, number>>` — queries collection_items joined with releases, classifies each record's rarity_score into gem tiers using SQL CASE expression (same thresholds as getGemTier), returns count per tier. Use Drizzle `db.execute(sql\`...\`)` with the CASE WHEN pattern from RESEARCH.md.

    Also export `getGemScoreForUser(userId: string): Promise<number>` — similar query but returns SUM of gem weights instead of counts.

    **3. Create `apps/web/src/components/ui/gem-badge.tsx`:**

    Match the exact structure from UI-SPEC.md:
    - Props: `{ score: number | null | undefined; showScore?: boolean; className?: string }`
    - Returns null if score is null/undefined
    - Uses `getGemTier()` and `getGemInfo()` from constants
    - Layout: `inline-flex items-center gap-1 font-mono text-[10px] font-semibold px-2 py-0.5 rounded-full border whitespace-nowrap`
    - Per-tier styling using Tailwind gem color classes (bg-gem-{tier}/10, text-gem-{tier}, border-gem-{tier}/25)
    - GemIcon inline component using Lucide icons: Hexagon for quartz, Gem for amethyst/emerald/ruby, Diamond for sapphire/diamond — all at size={12} strokeWidth={2} fill="currentColor" at 30% opacity
    - aria-label="{TierName} gem, weight {N}"
    - Hover effect CSS class per tier (no effect for quartz, gem-glow-purple for amethyst, gem-shimmer-green for emerald, gem-pulse-warm for ruby, gem-glow-blue+sparkle for sapphire, always-active prismatic for diamond)
    - Diamond badge gets class `animate-gem-prismatic` (always active, not hover-only)
    - All other tiers get hover-triggered classes via `group-hover` or `hover:` prefix
    - Optional showScore renders `x{weight}` at 70% opacity after tier name

    Add keyframe animations (exact CSS from UI-SPEC.md):
    - `gem-glow-purple`: box-shadow pulse 0 0 4px to 0 0 10px at oklch(0.55 0.22 290) with 0.2-0.4 alpha, 1.5s ease-in-out infinite
    - `gem-shimmer-green`: background-position slide -200% to 200%, 2s linear infinite; applied as linear-gradient with oklch(0.68 0.16 165 / 0.15) at 50%
    - `gem-pulse-warm`: box-shadow 0 0 4px to 0 0 14px at oklch(0.60 0.22 25) with 0.3-0.6 alpha, 1.2s ease-in-out infinite
    - `gem-glow-blue`: box-shadow 0 0 6px to 0 0 16px at oklch(0.58 0.20 260) with 0.3-0.5 alpha, 1.5s ease-in-out infinite
    - `gem-sparkle`: opacity/scale/rotate 0->1->0 over 1.2s ease-in-out infinite
    - `gem-prismatic`: hue-rotate 0->90->180->270->360 with brightness 1.1->1.2->1.3->1.2->1.1, 4s linear infinite
    - `gem-rainbow-edge`: border-color cycling through oklch hue 0->60->120->180->240->300->360, 3s linear infinite

    Wrap all gem animations in `@media (prefers-reduced-motion: reduce)` to disable them. Under reduced motion, Diamante gets static `border-color: oklch(0.70 0.20 290)` and subtle box-shadow.
