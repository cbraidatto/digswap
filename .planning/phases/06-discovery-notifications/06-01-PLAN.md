    **1. `searchRecords(term: string, limit = 20)`** (per D-04, DISC2-01):
    - Sanitize `term` by escaping `%`, `_`, `\` characters
    - Build pattern `%${sanitized}%`
    - Query `releases` table using Drizzle `db` client with `or(ilike(releases.title, pattern), ilike(releases.artist, pattern))`
    - Order by `sql\`COALESCE(${releases.rarityScore}, -1) DESC\``
    - Limit to `limit` rows
    - Collect matching release IDs
    - If no results, return empty array
    - Batch query `collectionItems` joined with `profiles` WHERE `releaseId IN (matchedIds)` to get owners
    - Each owner: `{ releaseId, userId, username, avatarUrl, conditionGrade }`
    - Group owners by releaseId using a Map
    - Return array of `{ id, title, artist, label, format, year, genre, rarityScore, coverImageUrl, owners: Owner[], ownerCount: number }`
    - Import: `db` from `@/lib/db`, `releases` from `@/lib/db/schema/releases`, `collectionItems` from `@/lib/db/schema/collections`, `profiles` from `@/lib/db/schema/users`, `sql, eq, or, ilike` from `drizzle-orm`

    **2. `browseRecords(genre: string | null, decade: string | null, limit = 20, offset = 0)`** (per D-06, D-07, DISC2-02):
    - Import `getDecadeRange` from `@/lib/collection/filters`
    - Build WHERE conditions array:
      - If `genre` is provided: `sql\`${releases.genre} @> ARRAY[${genre}]\`` (PostgreSQL array contains)
      - If `decade` is provided: use `getDecadeRange(decade)` to get `{ start, end }`, filter `releases.year >= start AND releases.year < end`
    - Query `releases` joined with `collectionItems` (left join or subquery) to get owner counts
    - Actually: query `collectionItems` inner-joined with `releases` (to only show records that exist in at least one user's collection)
    - Group by release, count distinct `userId` as `ownerCount`
    - Order by `rarityScore DESC NULLS LAST`
    - Limit and offset for pagination
    - Return array of `{ id, title, artist, label, format, year, genre, rarityScore, coverImageUrl, ownerCount }`

    **3. `getSuggestedRecords(userId: string, limit = 8)`** (per D-02, DISC2-04):
    - **Top genres query:** Get the user's top 3 genres by counting `collectionItems` joined with `releases`, grouping by `unnest(releases.genre)`, ordering by count DESC, limit 3
    - Query releases in those genres that the user does NOT own (LEFT JOIN collectionItems WHERE userId = currentUser AND collectionItems.id IS NULL), order by rarityScore DESC, limit `limit`
    - **Followed users' records query:** Get release IDs from `collectionItems` WHERE `userId IN (SELECT followingId FROM follows WHERE followerId = currentUser)` AND `releaseId NOT IN (SELECT releaseId FROM collectionItems WHERE userId = currentUser)`, joined with releases, order by rarityScore DESC, limit `limit`
    - Merge both result sets, deduplicate by release ID, return first `limit` items
    - Return array of `{ id, title, artist, label, format, year, genre, rarityScore, coverImageUrl, ownerCount }`
    - If user has no collection or no follows, return empty array (per specifics: "best effort")

"A digger"} has "${release.title}" by ${release.artist}\`, link: \`/perfil/${adderProfile?.username ?? ""}\`, read: false })`
