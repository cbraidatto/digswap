    --- ProposalBuilder.tsx (CLIENT component — "use client") ---

    Props:
    ```typescript
    interface ProposalBuilderProps {
      myItems: TradeableItem[];
      theirItems: TradeableItem[];
      targetUserId: string;
      targetUsername: string;
      isPremium: boolean;
      currentUserId: string;
    }
    ```

    State:
    - pendingItem: { item: TradeableItem; side: 'offer' | 'want' } | null — item awaiting quality declaration
    - offerBasket: { item: TradeableItem; declaredQuality: string; conditionNotes?: string }[]
    - wantBasket: { item: TradeableItem; declaredQuality: string; conditionNotes?: string }[]
    - message: string
    - isSubmitting: boolean
    - error: string | null

    maxItems = isPremium ? 3 : 1

    Handlers:
    - handleItemClick(item, side): set pendingItem = { item, side }
    - handleQualityConfirm(item, quality): add to appropriate basket, clear pendingItem
    - handleRemoveFromBasket(itemId, side): remove from basket
    - handleSubmit: call createProposalAction, on success router.push('/trades/' + tradeId), on error set error state

    Layout (responsive):
    - Mobile: stacked (offers above, wants below)
    - Desktop (lg): CSS grid 2 columns — left = CollectionColumn "Your Offers" (myItems), right = CollectionColumn "You Want" (theirItems)

    Below the two columns, a Proposal Summary section:
    - "Offer" list: offerBasket items with title + quality + remove button
    - "Want" list: wantBasket items with title + quality + remove button
    - Message textarea (optional)
    - Submit button: "Send Proposal" — disabled if offerBasket or wantBasket empty, loading state during submit
    - Tier badge: if !isPremium, show "Free tier: 1 item per side · Upgrade for 3:3 trades" note

    Free tier enforcement in UI: CollectionColumn receives isDisabled=true for each card when basket.length >= maxItems AND item not selected.

    QualityDeclarationModal: render below layout, open when pendingItem != null.

    Use useRouter from "next/navigation" for redirect after submit.
